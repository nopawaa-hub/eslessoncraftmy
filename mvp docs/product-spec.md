Product Overview

The Schedule Planner is the MVP core module of the AI Teacher Management Platform.

This module focuses ONLY on:

-Weekly timetable management
-Lesson scheduling
-Basic lesson workflow
-Lesson plan attachment
-Materials attachment (in PNG/JPG or any image format)
Reflection notes
Lightweight teacher organization

The system is NOT a full ERP.

The goal is to create:

-a clean operational teaching workspace
-a modern schedule planner
-an intuitive daily workflow system for teachers

The experience should feel:

-lightweight
-premium
-modern
-easy to understand
-low cognitive load
-practical for real-world school usage

--Primary Objective--

Enable teachers to:
-Create lesson plan using ai engine
-Create and customize their timetable
-Add teaching sessions
-Manage lesson-related resources
-Organize teaching workflow from one place

The schedule acts as the teacher’s operational hub.

--MVP Scope--
-Included-
-Weekly timetable
-Schedule customization
-Lesson blocks
-Drag and drop scheduling
-Lesson detail sidebar
-Attach lesson plans
-Upload materials
-Reflection notes
-Completion tracking
-Responsive UI

Excluded
-Advanced analytics
-Parent portal
-Student portal
-AI prediction systems
-Multi-school management
-Finance systems
-Attendance systems
-Enterprise reporting

User Persona
-Primary User-

Teacher

Characteristics:

-Busy
-Non-technical
-Needs fast workflow
-Wants visual clarity
-Needs organization
-Uses mobile/tablet frequently
-Core Philosophy

The system should feel:

-obvious to use
-visually calming
-operationally efficient
-frictionless
