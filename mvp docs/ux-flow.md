Main User Workflow

Teacher opens Schedule Planner 
↓ 
System loads weekly timetable 
↓ 
Teacher customizes school hours
↓ 
Teacher creates lesson blocks 
↓ 
Teacher drags lessons into correct time slots 
↓ 
Teacher clicks a lesson block 
↓ 
Sidebar opens 
↓ 
Teacher can:

add lesson plan from ai generated one or upload their own lesson plan
attach materials screenshot in any image format(optional)
write notes
add reminders
mark lesson completed 
↓ 
After lesson:
teacher writes reflection
uploads evidence (optional)
Schedule Creation Flow
Teacher opens settings
Sets:
school start time
school end time
lesson duration
recess time
System auto-generates timetable grid
Teacher adds recurring classes
Teacher edits timetable visually

Lesson Block Interaction Flow

Hover Lesson Block 
↓ 
Quick Actions Appear

edit
duplicate
move
delete 
↓ 
Click Lesson Block 
↓ 
Open Lesson Sidebar 
↓ 
Manage lesson session

Drag and Drop Flow

Teacher drags lesson 
↓ 
Teacher can resize block to adjust the time
↓ 
Grid snapping activates 
↓ 
Collision detection runs 
↓ 
If overlap: show warning 
↓ 
Else: auto-update lesson timing
